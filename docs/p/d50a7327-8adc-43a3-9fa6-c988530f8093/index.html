03. 决策树与随机森林 (Decision Trees & Random Forest)

🎯 核心考点​

  • 信息增益 (Information Gain)
  • 基尼指数 (Gini Index)
  • 剪枝 (Pruning)
  • 集成学习 (Ensemble Learning)
  • 随机性来源 (Randomness in Random Forest)

❓ 常见问题与答案​

1. 决策树如何划分特征？​

以信息增益最大为准：

其中 ( H(D) = -\sum_k p_k \log p_k )。

2. Gini 指数公式​

Gini(D)=1−∑kpk2Gini(D) = 1 - \sum_k p_k^2Gini(D)=1−k∑​pk2​

3. 如何防止决策树过拟合？​

  • 限制深度 (max_depth)
  • 剪枝 (post-pruning)
  • 限制最小样本数
  • 使用随机森林集成

4. 随机森林的两个随机性来源？​

  1. 样本随机 (Bootstrap Sampling)
  2. 特征随机 (Random Feature Subset)

5. 特征重要性 (Feature Importance)​

可通过信息增益或基尼下降量计算平均贡献。

💡 加分题​

  • 为什么随机森林不容易过拟合？

因为平均多个弱相关模型的预测可减少方差。

  • 决策树如何处理缺失值？

使用 surrogate split 或基于概率的分配策略。
