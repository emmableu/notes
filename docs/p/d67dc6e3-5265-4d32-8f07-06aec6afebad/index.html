第一章：强化学习问题（Sutton & Barto 中文版整理）

🎯 学习目标​

  • 理解强化学习（Reinforcement Learning, RL）的核心特征
  • 掌握 RL 与监督学习、无监督学习的区别
  • 熟悉 RL 的基本组成部分（agent、环境、策略、价值函数、奖励信号、模型）
  • 通过井字棋（Tic-Tac-Toe）示例理解 RL 中探索-利用、值函数、学习更新等核心概念
  • 概览强化学习的历史与发展脉络

📘 1. 强化学习问题简介​

我们从一个朴素的观察开始：人类如何学习？婴儿通过与环境互动、挥动手臂、观察反应，自主学习因果关系、目标达成方法等。这种“交互式学习”是我们理解世界的主要方式。

强化学习正是对这种学习方式的计算模拟。我们从 AI 工程师的角度来研究——如何让机器通过与环境的互动，学习达成目标的能力。

强化学习方法涉及：

  • 理论建模（如马尔可夫决策过程）
  • 学习策略（如何行动）
  • 评估方法（数学分析或实验）

🧩 1.1 什么是强化学习？​

强化学习（Reinforcement Learning）兼具三重身份：

  1. 一种问题类型：学习如何行动以最大化奖励
  2. 一类方法集合：解决这类问题的算法（如 Q-learning、PPO）
  3. 一个研究领域：研究这一问题与方法的学科

强化学习的三大特征：

特征解释
闭环系统（Closed-loop）行动影响环境，环境反馈影响未来感知
无指导反馈（No direct instruction）无“正确答案”，agent 需通过尝试获得奖励最大化策略
延迟反馈（Delayed consequences）奖励可能延迟多步才显现，行为影响远期结果

强化学习是第三种学习范式，区别于：

  • 监督学习（Supervised Learning）
  • 无监督学习（Unsupervised Learning）

强化学习的关键问题：探索（exploration）与利用（exploitation）权衡

🤖 1.2 示例与动机​

强化学习应用广泛，以下是经典示例：

  • ♟️ 国际象棋大师的落子决策（融合直觉和规划）
  • 🏭 工业控制器在线调节石化工厂运行策略
  • 🦌 小鹿出生数分钟后奔跑，行为学习迅速
  • 🤖 机器人在垃圾回收与返回充电站之间决策
  • 🧍‍♂️ Phil 做早餐涉及多重子目标（如拿碗、倒牛奶）

这些例子都体现了强化学习的核心特征：

  • 明确目标（目标导向）
  • 行动-环境交互（agent 可以影响环境）
  • 奖励延迟与规划需求
  • 环境不可预测，需要 agent 具备适应能力

🧱 1.3 强化学习的基本组成要素​

要素说明
策略（Policy）agent 在给定状态下的行为规则（可以是函数、查找表、搜索机制等）
奖励（Reward）表征即时好坏的数值信号（生理类比：愉悦 vs. 痛苦）
价值函数（Value Function）衡量长期好坏：从当前状态出发的期望奖励总和
环境模型（Model）预测未来状态与奖励的机制，用于规划（可选）

奖励 是“目标”；价值函数是“手段”；策略是“选择的具体行为”；模型用于模拟未来。

🎲 1.4 范围与限制​

  • 本书强调基于值函数的 RL 方法
  • 与进化方法（如遗传算法）对比：后者对每个完整策略评估，不关注中间过程
  • 进化方法可能效率低，忽略了状态-动作结构
  • 我们关注的是：在交互中逐步学习策略（可结合监督、模型）

🎮 1.5 井字棋强化学习示例​

通过井字棋游戏展示强化学习的核心流程：

🌟 设定：​

  • 我们是 X 方，对手是不完美玩家
  • 状态空间：所有可能棋局
  • 初始估值表：胜利状态=1，失败/平局=0，其他=0.5（初始化）

🎯 策略：​

  • 贪婪策略（Greedy）：选择估值最高的下一步
  • 探索动作（Exploratory）：偶尔随机选择动作，访问未知状态

📉 更新规则：​

只在贪婪动作上学习，用时间差分方法更新状态值：

V(s)←V(s)+α[V(s′)−V(s)]V(s) ← V(s) + α [V(s') - V(s)] V(s)←V(s)+α[V(s′)−V(s)]

其中 α 是学习率（step-size parameter）

如图所示：除去探索动作，每一步都进行“值回传”（backup），更新当前状态的估值

🧠 学习结果：​

  • 如果 α 合理衰减，最终能学到最优策略
  • 学到的策略对缓慢变化的对手也具鲁棒性

🔁 强化学习 vs. 进化方法​

方法特点
进化方法（EA）整体评估策略，不关注中间状态，样本效率低
值函数方法（RL）利用每步状态信息、可边玩边学、能针对关键状态做更新

🧬 1.6 总结​

强化学习是：

  • 目标导向学习与决策的计算方法
  • 基于状态-动作-奖励的形式建模
  • 与监督/无监督学习并列的独立学习范式
  • 重视 值函数估计，以更高效搜索最优策略

📜 1.7 强化学习的历史（简要）​

分支内容
试错学习（Trial-and-error）来源于心理学实验（Thorndike、Skinner），强调强化与行为选择
最优控制（Optimal Control）Bellman 提出动态规划，建立 MDP 形式化建模
时间差分学习（TD）Samuel（1959）checkers，Sutton&Barto 提出 TD(λ)、Actor-Critic 架构等

强化学习融合了上述三条路径，形成现代强化学习：

  • 动态规划 + TD 学习 + 值函数近似 + 深度学习 → 如 AlphaGo、TD-Gammon 等

🔍 引导问题与API说明

功能描述示例输入示例输出
one_round一次出手，先O再X当前状态+空格索引获胜方或NOWIN
check_win检查当前棋盘是否有胜方棋盘状态'X' / 'O' / 'NOWIN'

update_policy_dict根据当前episode更新所有state的价值胜者、状态历史、策略表更新后的策略和TD误差
calculate_win_rate训练N轮游戏并记录结果和TD误差训练轮数胜率+误差日志

🧪 随机策略实现（baseline）

import random
from collections import defaultdict

no_win_key = "NOWIN"

def print_state(state):
    print("-----------")
    for idx in range(3):
        print("|", " ".join(state[idx * 3 : (idx + 1) * 3]), "|")
    print("-----------")

def one_round(cur_arr, state):
    o_pos_idx = random.randint(0, len(cur_arr) - 1)
    o_pos = cur_arr[o_pos_idx]
    state[o_pos] = 'O'
    win = check_win(cur_arr, state)
    if win != no_win_key:
        return win
    del cur_arr[o_pos_idx]
    x_pos_idx = random.randint(0, len(cur_arr) - 1)
    x_pos = cur_arr[x_pos_idx]
    state[x_pos] = 'X'
    del cur_arr[x_pos_idx]
    win = check_win(cur_arr, state)
    if win != no_win_key:
        return win
    return no_win_key

def check_win(cur_arr, state, final_round=False):
    if final_round:
        state[cur_arr[0]] = 'O'
    win_pos_lst = [[0,1,2], [3,4,5], [6,7,8],
                   [0,3,6], [1,4,7], [2,5,8],
                   [0,4,8], [2,4,6]]
    for win_pos in win_pos_lst:
        if state[win_pos[0]] == state[win_pos[1]] == state[win_pos[2]]:
            if state[win_pos[0]] == 'X':
                return "X"
            if state[win_pos[0]] == 'O':
                return "O"
    return 'NOWIN'

def full_game():
    cur_arr = list(range(9))
    state = [" " for _ in range(9)]
    for _ in range(4):
        win = one_round(cur_arr, state)
        if win != no_win_key:
            return win
    return check_win(cur_arr, state, final_round=True)

def calculate_win_rate():
    res = defaultdict(int)
    for _ in range(1000):
        res[full_game()] += 1
    return res

res = calculate_win_rate()
print(res)
# defaultdict(<class 'int'>, {'O': 581, 'X': 309, 'NOWIN': 110})

🧠 基于强化学习（TD 更新）的策略

import math
import random
from collections import defaultdict
import numpy as np
import torch
import torch.nn.functional as F
import matplotlib.pyplot as plt

no_win_key = "NOWIN"

def print_state(state):
    print("-----------")
    for idx in range(3):
        print("|", " ".join(state[idx * 3: (idx + 1) * 3]), "|")
    print("-----------")

def get_x_pos_idx(cur_arr, state, policy_dict):
    candidate_prob_lst = []
    for idx in range(len(cur_arr)):
        x_pos = cur_arr[idx]
        next_state = state.copy()
        next_state[x_pos] = "X"
        state_str = "".join(next_state)
        prob = policy_dict[state_str]
        candidate_prob_lst.append(prob)
    probs_tensor = torch.tensor(candidate_prob_lst)
    top_idx = torch.argmax(probs_tensor)
    return top_idx

def one_round(cur_arr, state, policy_dict, state_history):
    o_pos_idx = random.randint(0, len(cur_arr) - 1)
    o_pos = cur_arr[o_pos_idx]
    state[o_pos] = 'O'
    state_history.append(''.join(state))
    win = check_win(cur_arr, state)
    if win != no_win_key:
        return win, state_history
    del cur_arr[o_pos_idx]
    x_pos_idx = get_x_pos_idx(cur_arr, state, policy_dict)
    x_pos = cur_arr[x_pos_idx]
    state[x_pos] = 'X'
    del cur_arr[x_pos_idx]
    state_history.append(''.join(state))
    win = check_win(cur_arr, state)
    if win != no_win_key:
        return win, state_history
    return no_win_key, state_history

def check_win(cur_arr, state, final_round=False):
    if final_round:
        state[cur_arr[0]] = 'O'
    win_pos_lst = [[0,1,2], [3,4,5], [6,7,8],
                   [0,3,6], [1,4,7], [2,5,8],
                   [0,4,8], [2,4,6]]
    for win_pos in win_pos_lst:
        if state[win_pos[0]] == state[win_pos[1]] == state[win_pos[2]]:
            if state[win_pos[0]] == 'X':
                return "X"
            if state[win_pos[0]] == 'O':
                return "O"
    return 'NOWIN'

def full_game(policy_dict):
    cur_arr = list(range(9))
    state = [" " for _ in range(9)]
    state_history = []
    for _ in range(4):
        win, state_history = one_round(cur_arr, state, policy_dict, state_history)
        if win != no_win_key:
            return win, state_history
    return check_win(cur_arr, state, final_round=True), state_history

def update_policy_dict(win, state_history, policy_dict, log_dict):
    final_reward = {'X': 1, 'O': 0, 'NOWIN': 0.5}[win]
    prev_reward = final_reward
    td_err_lst = []
    for state in reversed(state_history):
        td_err = prev_reward - policy_dict[state]
        policy_dict[state] += 0.1 * td_err
        td_err_lst.append(td_err)
        prev_reward = policy_dict[state]
    log_dict['td_err'].append(max(td_err_lst))
    return policy_dict, log_dict

def calculate_win_rate(rounds, policy_dict=None):
    res = defaultdict(int)
    if policy_dict is None:
        policy_dict = defaultdict(lambda: 0.5)
    log_dict = defaultdict(list)
    for _ in range(rounds):
        win, state_history = full_game(policy_dict)
        policy_dict, log_dict = update_policy_dict(win, state_history, policy_dict, log_dict)
        res[win] += 1
    return res, log_dict

rounds = 1000
res, log_dict = calculate_win_rate(rounds)
print(res)
# defaultdict(<class 'int'>, {'O': 283, 'X': 585, 'NOWIN': 132})

📈 TD Error Scatter Plot + Moving Average

def plot_loss():
    def moving_average(data, window_size=20):
        return np.convolve(data, np.ones(window_size)/window_size, mode='valid')

    td_err = log_dict['td_err']
    ma_td_err = moving_average(td_err, window_size=20)

    plt.figure(figsize=(10, 4))
    plt.scatter(range(rounds), td_err, s=10, alpha=0.4, label='TD Error')
    plt.plot(range(len(ma_td_err)), ma_td_err, color='red', label='Moving Avg (20)')
    plt.xlabel("Episode")
    plt.ylabel("TD Error")
    plt.title("TD Error Scatter and Moving Average")
    plt.legend()
    plt.grid(True)
    plt.tight_layout()
    plt.show()

plot_loss()
