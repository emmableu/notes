概率与似然（HTML 版） + MOM vs MLE

🐭 老鼠体重估计：样本均值（MOM） vs 最大似然估计（MLE）​

问题​

给定一组老鼠体重观测值，估计总体均值 μ\muμ 与方差 σ2\sigma^2σ2。比较两种方法：

  1. 样本均值 / 矩估计 (Method of Moments, MOM)
  2. 最大似然估计 (MLE)（假设体重独立同分布且服从高斯分布）

数据与假设​

  • 观测数据（单位：g）： y=[180,  190,  175,  200,  185]y = [180,\; 190,\; 175,\; 200,\; 185]y=[180,190,175,200,185]
  • 记 n=5n=5n=5，样本和为 ∑yi=930\sum y_i = 930∑yi​=930

方法 A：样本均值 / 矩估计（MOM）​

计算思路​

  • 一阶矩匹配：用 样本均值 作为总体均值的估计 μ^MOM=yˉ=1n∑i=1nyi\hat\mu_{\text{MOM}} = \bar{y} = \frac{1}{n}\sum_{i=1}^n y_iμ^​MOM​=yˉ​=n1​i=1∑n​yi​
  • 二阶中心矩匹配的常见无偏估计（用于比较）： σ^unb2=1n−1∑i=1n(yi−yˉ)2\hat\sigma^2_{\text{unb}}=\frac{1}{n-1}\sum_{i=1}^n (y_i-\bar{y})^2σ^unb2​=n−11​i=1∑n​(yi​−yˉ​)2

具体计算​

  • 样本均值： yˉ=9305=186\bar{y} = \frac{930}{5} = 186yˉ​=5930​=186
  • 离差：(−6,  +4,  −11,  +14,  −1)(-6,\; +4,\; -11,\; +14,\; -1)(−6,+4,−11,+14,−1)
  • 离差平方和（RSS w.r.t. yˉ\bar{y}yˉ​）： ∑(yi−yˉ)2=36+16+121+196+1=370\sum (y_i-\bar{y})^2 = 36+16+121+196+1 = 370∑(yi​−yˉ​)2=36+16+121+196+1=370
  • 无偏方差估计（作参考）： σ^unb2=3705−1=92.5\hat\sigma^2_{\text{unb}} = \frac{370}{5-1} = 92.5σ^unb2​=5−1370​=92.5
  • 无偏标准差：92.5≈9.62\sqrt{92.5}\approx 9.6292.5​≈9.62

结果（MOM）​

  • μ^MOM=186\hat\mu_{\text{MOM}} = 186μ^​MOM​=186
  • （参考）σ^unb2=92.5\hat\sigma^2_{\text{unb}} = 92.5σ^unb2​=92.5，σ^unb≈9.62\hat\sigma_{\text{unb}}\approx 9.62σ^unb​≈9.62

方法 B：最大似然估计（MLE, 高斯假设）​

模型与似然​

  • 假设 yi∼iidN(μ,σ2)y_i \overset{iid}{\sim} \mathcal{N}(\mu,\sigma^2)yi​∼iidN(μ,σ2)
  • 概率密度公式：
f(x∣μ,σ2)=12πσ2exp⁡ ⁣(−(x−μ)22σ2),−∞<x<∞f(x \mid \mu, \sigma^2) = \frac{1}{\sqrt{2\pi\sigma^2}} \exp\!\left(-\frac{(x-\mu)^2}{2\sigma^2}\right), \quad -\infty < x < \inftyf(x∣μ,σ2)=2πσ2​1​exp(−2σ2(x−μ)2​),−∞<x<∞
  • 对数似然： ℓ(μ,σ2)=−n2log⁡(2πσ2)−12σ2∑i=1n(yi−μ)2\ell(\mu,\sigma^2) = -\frac{n}{2}\log(2\pi\sigma^2) -\frac{1}{2\sigma^2}\sum_{i=1}^n (y_i-\mu)^2ℓ(μ,σ2)=−2n​log(2πσ2)−2σ21​i=1∑n​(yi​−μ)2

对 μ\muμ 最大化（固定 σ2\sigma^2σ2）​

  • 一阶条件： ∂ℓ∂μ=1σ2∑i=1n(yi−μ)=0  ⇒  μ^MLE=yˉ=186\frac{\partial \ell}{\partial \mu} = \frac{1}{\sigma^2}\sum_{i=1}^n (y_i-\mu) = 0 \;\Rightarrow\; \hat\mu_{\text{MLE}} = \bar{y} = 186∂μ∂ℓ​=σ21​i=1∑n​(yi​−μ)=0⇒μ^​MLE​=yˉ​=186

对 σ2\sigma^2σ2 最大化（代入 μ^\hat\muμ^​）​

  • 一阶条件： ∂ℓ∂σ2=−n2⋅1σ2+12⋅∑(yi−μ^)2(σ2)2=0\frac{\partial \ell}{\partial \sigma^2} = -\frac{n}{2}\cdot\frac{1}{\sigma^2} + \frac{1}{2}\cdot\frac{\sum (y_i-\hat\mu)^2}{(\sigma^2)^2} = 0∂σ2∂ℓ​=−2n​⋅σ21​+21​⋅(σ2)2∑(yi​−μ^​)2​=0 ⇒σ^MLE2=1n∑i=1n(yi−μ^)2=3705=74\Rightarrow\quad \hat\sigma^2_{\text{MLE}}=\frac{1}{n}\sum_{i=1}^n (y_i-\hat\mu)^2 = \frac{370}{5} = 74⇒σ^MLE2​=n1​i=1∑n​(yi​−μ^​)2=5370​=74
  • MLE 标准差：σ^MLE=74≈8.60\hat\sigma_{\text{MLE}}=\sqrt{74}\approx 8.60σ^MLE​=74​≈8.60

结果（MLE）​

  • μ^MLE=186\hat\mu_{\text{MLE}} = 186μ^​MLE​=186
  • σ^MLE2=74\hat\sigma^2_{\text{MLE}} = 74σ^MLE2​=74，σ^MLE≈8.60\hat\sigma_{\text{MLE}}\approx 8.60σ^MLE​≈8.60

并列对比​

指标MOM（样本均值/无偏方差）MLE（高斯假设）
均值估计 μ^\hat\muμ^​186186
方差估计 σ^2\hat\sigma^2σ^292.5（无偏，除以 n−1n-1n−1）74（MLE，除以 nnn）
标准差≈9.62\approx 9.62≈9.62≈8.60\approx 8.60≈8.60
备注不依赖显式分布假设；均值天然一致依赖高斯假设；均值与 MOM 重合，方差因分母不同而偏小

关键结论与差异​

  1. 均值估计一致：在正态模型下，MLE 的 μ^\hat\muμ^​ 与 样本均值（MOM） 完全相同。
  2. 方差估计不同：
    • MLE 用分母 nnn，是有偏但似然意义下最优；
    • 无偏方差 用分母 n−1n-1n−1，在有限样本下对总体方差更准确。
  3. 方法取舍：
    • 若你需要严格的统计推断（如置信区间/检验）且样本不大，常用无偏方差；
    • 若坚持似然框架或后续做极大似然/贝叶斯扩展，MLE 一致性与可组合性更好。
