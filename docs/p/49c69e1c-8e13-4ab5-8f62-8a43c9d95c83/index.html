01. 线性回归 (Linear Regression)

🎯 核心考点​

  • 假设函数 (Hypothesis function)
  • 最小二乘法 (Least Squares)
  • 损失函数 (Loss Function)
  • 正则化 (Regularization)
  • 偏差–方差权衡 (Bias–Variance Tradeoff)

❓ 常见问题与答案​

1. 线性回归的假设函数是什么？​

假设目标变量 ( y ) 与输入变量 ( x ) 满足：

y^=hθ(x)=θ0+θ1x1+⋯+θnxn\hat{y} = h_\theta(x) = \theta_0 + \theta_1 x_1 + \cdots + \theta_n x_ny^​=hθ​(x)=θ0​+θ1​x1​+⋯+θn​xn​

其中 ( \theta ) 为参数向量。

2. 损失函数是什么？​

使用最小化平方误差 (Mean Squared Error, MSE)：

J(θ)=12m∑i=1m(hθ(x(i))−y(i))2J(\theta) = \frac{1}{2m} \sum_{i=1}^m (h_\theta(x^{(i)}) - y^{(i)})^2J(θ)=2m1​i=1∑m​(hθ​(x(i))−y(i))2

加上「1/2」是为了在求导时消掉平方项带来的 2， 使梯度公式更整洁

3. 如何推导梯度下降 (Gradient Descent)？​

对参数 ( \theta_j ) 求偏导：

∂J(θ)∂θj=1m∑i=1m(hθ(x(i))−y(i))xj(i)\frac{\partial J(\theta)}{\partial \theta_j} = \frac{1}{m} \sum_{i=1}^m (h_\theta(x^{(i)}) - y^{(i)})x_j^{(i)}∂θj​∂J(θ)​=m1​i=1∑m​(hθ​(x(i))−y(i))xj(i)​

更新规则：

θj:=θj−α∂J(θ)∂θj\theta_j := \theta_j - \alpha \frac{\partial J(\theta)}{\partial \theta_j}θj​:=θj​−α∂θj​∂J(θ)​

4. L1 与 L2 正则化的区别？​

正则化数学形式几何意义特点
L1 (Lasso)( \lambda \sum\theta_i)
L2 (Ridge)( \lambda \sum \theta_i^2 )圆形约束防止权重过大，平滑

5. 为什么标准化 (Normalization) 很重要？​

标准化可以让所有特征处于同一数量级，避免梯度下降震荡，收敛更快。

💡 加分题​

  • 为什么正则化能防止过拟合？

因为它限制了模型复杂度（较小的参数空间），降低方差 (variance)，提高泛化能力。

  • 如何判断模型是否欠拟合？

训练误差和验证误差都高 → 模型太简单，可增加特征或减少正则化。
