强化学习完全讲义：Sutton & Barto 第二版 第四章 动态规划 (Dynamic Programming)

编者：Emma Wang 整理教学版
参考书：《Reinforcement Learning: An Introduction (2nd Edition)》by Richard S. Sutton & Andrew G. Barto
语言：Python
篇幅：约 20,000+ 字（含完整代码 + 图表渲染）

🧭 一、章节总览​

动态规划（Dynamic Programming, DP）是强化学习中第一个具有系统化理论的求解方法。它在已知环境模型的情况下，能够精确求解最优策略（Optimal Policy）。虽然在实际强化学习（例如 RLHF）中，我们往往无法直接使用 DP，但理解 DP 是掌握 RL 的数学基础。

DP = 明确模型 + 递归价值更新 + 策略改进。

📘 二、核心概念对照表​

概念英文名数学定义直觉理解
状态State (s)环境的一个描述游戏中的“当前局面”
动作Action (a)智能体可执行操作向上、下、左、右等选择
策略Policy (π)(π(as)) 表示在状态 s 下执行动作 a 的概率行为模式
状态价值State-Value (v_π)(v_π(s) = E[R_tS_t = s])状态好坏的打分
动作价值Action-Value (q_π)(q_π(s,a) = E[R_tS_t = s, A_t = a])选这个动作能带来多大期望收益
模型Model(p(s',rs,a))环境的动力学：状态转移与奖励机制

💡 三、Bellman 方程（Bellman Equation）​

Bellman 方程是 DP 的灵魂。

✅ 状态价值函数形式​

vπ(s)=∑aπ(a∣s)∑s′,rp(s′,r∣s,a)[r+γvπ(s′)]v_π(s) = \sum_a π(a|s) \sum_{s',r} p(s',r|s,a)[r + γ v_π(s')]vπ​(s)=a∑​π(a∣s)s′,r∑​p(s′,r∣s,a)[r+γvπ​(s′)]

这表示：一个状态的价值 = 所有动作下的加权期望值（即“即时奖励 + 折扣未来价值”）。

✅ 最优价值函数形式​

V∗(s)=max⁡a∑s′,rp(s′,r∣s,a)[r+γV∗(s′)]V_*(s) = \max_a \sum_{s',r} p(s',r|s,a)[r + γ V_*(s')]V∗​(s)=amax​s′,r∑​p(s′,r∣s,a)[r+γV∗​(s′)]

这就是强化学习中所有算法的共同祖先。无论是 Q-learning、PPO 还是 RLHF，其本质都在逼近这个方程。

🧱 四、Gridworld 网格世界（完整讲解）​

我们从 Sutton & Barto 书中经典的 5x5 Gridworld 开始。

🧩 环境定义​

  • 环境是 5x5 方格。
  • 每个格子是一个状态 (i, j)。
  • 每次可以执行四个动作：上、下、左、右。
  • 如果移动出边界，则奖励 -1，回到原位。
  • 特殊传送门：
    • A → A' 奖励 +10
    • B → B' 奖励 +5
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

gamma = 0.9
grid_size = 5
A, A_ = (0, 1), (4, 1)
B, B_ = (0, 3), (2, 3)
actions = ['up', 'down', 'left', 'right']

def step(i, j, action):
    if (i, j) == A:
        return A_, 10
    if (i, j) == B:
        return B_, 5

    moves = {'up': (-1, 0), 'down': (1, 0), 'left': (0, -1), 'right': (0, 1)}
    di, dj = moves[action]
    ni, nj = i + di, j + dj

    if 0 <= ni < grid_size and 0 <= nj < grid_size:
        return (ni, nj), 0
    else:
        return (i, j), -1

v = np.zeros((grid_size, grid_size))

for k in range(1000):
    new_v = np.zeros_like(v)
    for i in range(grid_size):
        for j in range(grid_size):
            val = 0
            for a in actions:
                (ni, nj), r = step(i, j, a)
                val += 0.25 * (r + gamma * v[ni, nj])
            new_v[i, j] = val
    if np.max(np.abs(v - new_v)) < 1e-4:
        break
    v = new_v

sns.heatmap(v, annot=True, fmt=".1f", cmap="coolwarm")
plt.title('Policy Evaluation Value Function (Gridworld)')
plt.show()

🔁 五、策略迭代 (Policy Iteration)​

Policy Iteration 由两步交替执行： 1️⃣ Policy Evaluation （在当前策略下计算价值） 2️⃣ Policy Improvement （根据价值函数更新策略）

def policy_evaluation(policy, gamma=0.9, theta=1e-4):
    v = np.zeros((grid_size, grid_size))
    while True:
        delta = 0
        for i in range(grid_size):
            for j in range(grid_size):
                v_old = v[i, j]
                val = 0
                for a in actions:
                    (ni, nj), r = step(i, j, a)
                    val += policy[i, j, a] * (r + gamma * v[ni, nj])
                v[i, j] = val
                delta = max(delta, abs(v_old - val))
        if delta < theta:
            break
    return v

def policy_improvement(v, gamma=0.9):
    policy = np.zeros((grid_size, grid_size, len(actions)))
    stable = True
    for i in range(grid_size):
        for j in range(grid_size):
            q_vals = []
            for a in actions:
                (ni, nj), r = step(i, j, a)
                q_vals.append(r + gamma * v[ni, nj])
            best_a = np.argmax(q_vals)
            for a in range(len(actions)):
                policy[i, j, a] = 1 if a == best_a else 0
    return policy

policy = np.ones((grid_size, grid_size, len(actions))) / len(actions)
for _ in range(10):
    v = policy_evaluation(policy)
    policy = policy_improvement(v)

plt.figure(figsize=(6,5))
sns.heatmap(v, annot=True, fmt=".2f", cmap="YlGnBu")
plt.title('Policy Iteration Final Value Function')
plt.show()

⚡ 六、价值迭代 (Value Iteration)​

价值迭代将策略评估与改进合并为一步：

V(s)←max⁡a∑s′,rp(s′,r∣s,a)[r+γV(s′)]V(s) \leftarrow \max_a \sum_{s',r} p(s',r|s,a)[r + γV(s')]V(s)←amax​s′,r∑​p(s′,r∣s,a)[r+γV(s′)] 
v = np.zeros((grid_size, grid_size))
for it in range(200):
    new_v = np.zeros_like(v)
    for i in range(grid_size):
        for j in range(grid_size):
            values = []
            for a in actions:
                (ni, nj), r = step(i, j, a)
                values.append(r + gamma * v[ni, nj])
            new_v[i, j] = max(values)
    if np.max(np.abs(v - new_v)) < 1e-4:
        break
    v = new_v

sns.heatmap(v, annot=True, fmt=".2f", cmap="viridis")
plt.title('Value Iteration Value Function')
plt.show()

🧠 七、RLHF 对应思想​

在 RLHF 中，我们无法直接获得 (p(s',r|s,a))，但：

  • 奖励模型 (Reward Model) 相当于学习了一个 (\hat{r}(s,a))。
  • 策略优化 (Policy Optimization) 对应于 Policy Improvement。
  • PPO 的目标函数中 Advantage 项（A_t）正是 Bellman backup 的近似： (A_t = R_t + γV(s_{t+1}) - V(s_t))。

RLHF 本质上是使用神经网络逼近 DP 的计算过程。

🧩 八、LeetCode 类比问题​

🧮 LeetCode 64 最小路径和​

Bellman 等价式：

V(i,j)=grid[i][j]+min⁡(V(i+1,j),V(i,j+1))V(i,j) = grid[i][j] + \min(V(i+1,j), V(i,j+1))V(i,j)=grid[i][j]+min(V(i+1,j),V(i,j+1)) 
def minPathSum(grid):
    m, n = len(grid), len(grid[0])
    dp = [[0]*n for _ in range(m)]
    dp[-1][-1] = grid[-1][-1]
    for i in reversed(range(m)):
        for j in reversed(range(n)):
            if i == m-1 and j == n-1:
                continue
            down = dp[i+1][j] if i+1 < m else float('inf')
            right = dp[i][j+1] if j+1 < n else float('inf')
            dp[i][j] = grid[i][j] + min(down, right)
    return dp[0][0]

这与 Bellman backup 完全一致，只是最小化路径损失。

🧠 LeetCode 174 地下城游戏​

骑士必须保持健康值 > 0。

Bellman 倒推：

HP(i,j)=max⁡(1,min⁡(HP(i+1,j),HP(i,j+1))−dungeon[i][j])HP(i,j) = \max(1, \min(HP(i+1,j), HP(i,j+1)) - dungeon[i][j])HP(i,j)=max(1,min(HP(i+1,j),HP(i,j+1))−dungeon[i][j])

这也是典型的 DP → Bellman backup。

📊 九、收敛可视化​

我们绘制价值函数随迭代次数的变化：

v = np.zeros((grid_size, grid_size))
history = []
for it in range(200):
    new_v = np.zeros_like(v)
    for i in range(grid_size):
        for j in range(grid_size):
            values = []
            for a in actions:
                (ni, nj), r = step(i, j, a)
                values.append(r + gamma * v[ni, nj])
            new_v[i, j] = max(values)
    diff = np.mean(np.abs(v - new_v))
    history.append(diff)
    if diff < 1e-4:
        break
    v = new_v

plt.plot(history)
plt.title('Value Function Convergence (Value Iteration)')
plt.xlabel('Iterations')
plt.ylabel('Mean ΔV')
plt.show()
